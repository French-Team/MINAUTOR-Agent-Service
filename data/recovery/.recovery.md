# Reprise après redémarrage — Restauration des agents

## Contexte

Le dossier `.agents/` est verrouillé par Windows (état fantôme EPERM)
suite à des opérations filesystem intensives. Un redémarrage du système
est nécessaire pour libérer le verrou.

Après redémarrage, lancer Minautor Agents Service via `npm run final` puis demander :

> Recrée tous les agents dans .agents/

(ou exécuter directement les commandes de la section Procédure ci-dessous)

---

## Agents à recréer (9 au total)

> **Note** : Les types `ToolConfig`, `SelfCorrection` et `Guardian` ci-dessous
> correspondent exactement à l'interface `AgentDefinition` dans
> `src/types/agent-definition.ts`.

## Procédure de reprise

Une fois l'ordinateur redémarré :

```bash
# 1. Vérifier que le dossier est libéré
ls -la .agents/

# 2. Lancer le service
npm run final

# 3. Dans l'interface CLI, demander (option 2 = Start session → choisir Alice) :
#    "Recrée tous les agents dans .agents/ à partir de .recovery.md"
```

Ou bien, si l'interface CLI est déjà ouverte avec Alice, simplement demander :

> Restaure tous les agents depuis le fichier .recovery.md
